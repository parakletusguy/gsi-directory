Compact status / count pill. Use for source health, run status, region counts.

```jsx
<Badge tone="success" dot>Complete</Badge>
<Badge tone="warning" dot>In progress</Badge>
<Badge tone="danger" variant="outline">Failed</Badge>
<Badge tone="brand">1,284</Badge>
<Badge tone="info" variant="solid">API</Badge>
```

- **tone**: `neutral` · `brand` · `success` · `warning` · `danger` · `info`
- **variant**: `soft` (tinted, default) · `solid` · `outline`
- `dot` adds a leading status dot; `size` `sm`|`md`.
